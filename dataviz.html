Data Visualizations

United States Demographics

It is an interactive choropleth map of the United States, showing demographic data from 2015 American Community Survey .


On mouseover, each state displays summarized information about the state according to each category. The bar chart complements the information by offering additional data on either median value of all states or counties, or the distribution of data in the state or county. It is possible to switch between a state map and county map, based on what the user is looking for.


(For building the choropleth map: Codes from U.S. States TopoJSON and U.S. Counties TopoJSONwere used for rendering the base map.)

General Social Survey (1972-2014)

It is a visualization of General Social Survey data from 1972-2014. The process involved trimming down the data, designing, and coding the visualization. As the data was big in size (a csv file around 650MB), I used R to create a new csv files with variables to focus on in the visualization.


The objective was to explore different visualization designs that can give context to the data and stimulate insights.
